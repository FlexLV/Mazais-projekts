# Projekta nosaukums
Tetris
## Apraksts
Spēle tetris kas būs vizuāla un to varēs spēlēt noteikumi tādi paši kā tetris
## Galvenās funkcijas
	- Spēlēt spēli
	- Kustināt objektu
## Izmantotās tehnoloģijas
	- C++
	- SFML library
## Izmantotie avoti
